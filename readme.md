


### Technical Test  Frontend MyULibrary

#### Requirements
* NPM > 8.*
* Node >= 18.*



### Installation
* cp .env.example .env
* change `VITE_API_URL` if is needed (in case backend server is working on another port or domain)
* npm install
* npm run dev
